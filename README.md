# elm-lights-out-state-in-server

## Compile
elm-make src/Main.elm

## Run server
git clone https://github.com/trikitrok/lights-out-server-compojure.git
cd elm-lights-out-state-in-server
lein ring server

## Run client
cd elm-lights-out-state-in-server
open index.html
